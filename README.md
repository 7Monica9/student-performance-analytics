# 📊 Student Performance Analytics: Data-Informed EdTech PoC

🎯 **Bridging the gap between raw student data and actionable pedagogical insights.**

This intelligent dashboard is a professional **Proof of Concept (PoC)** designed for the EdTech and Instructional Design space. It demonstrates how data-driven analytics and Generative AI can empower educators to monitor progress and intervene early.

---

## 🚀 Core Capabilities

- **📈 Average Score Trends:** Track the pulse of the class with weekly mean score visualizations.
- **📊 Distribution Analysis:** Understand the spread of performance across the entire cohort.
- **⚠️ Risk Assessment:** A smart tracking table that automatically flags students with declining performance trends.
- **🤖 AI Pedagogical Insights:** One-click generation of 3–5 actionable teaching strategies powered by **Google Gemini 1.5 Flash**.

---

## ✨ Key Features

### 📋 Intelligent Data Handling
- **Synthetic Data Engine:** Generates realistic, privacy-safe data for 30 mock students over a 6-month period.
- **Multi-Subject Support:** Optimized for Math and English performance tracking.
- **Detailed Metrics:** Includes Student Name, Date, Subject, Score (0–100), and Engagement Level (1–5).

### 🎨 Premium UI/UX
- **Glassmorphism Design:** A clean, minimalist interface inspired by modern SaaS aesthetics.
- **Interactive Filters:** Sidebar controls to drill down into specific subjects or individual learners.
- **Responsive Charts:** High-fidelity visualizations powered by Altair.

### 🧠 AI Logic
- **Smart Fallback:** If a Gemini API key isn't provided, the system gracefully switches to a built-in pedagogical logic engine to ensure a seamless demo experience.
